A03:2021
Description (what to look for)
SQL injection vulnerabilities arise when user-controllable data is incorporated into database SQL queries in an unsafe manner. Test all input fields, headers, and URL parameters for injection points. Use tools such as sqlmap to automate discovery.
Default remediation
Use parameterized queries (prepared statements) for all database access. Validate and sanitize all user input server-side. Apply the principle of least privilege to database accounts.
